<h1>Projeto Para Aula</h1>

<h2>Iniciar Api</h2>

<h3>Instalar as Bibliotecas</h3>

```bash
    npm install
```
<h3>Criar o Arquivo .env</h3>

```bash
    DATABASE_URL="URL DO BANCO DE DADOS MONGODB"
```

<h3>Criar Tableas no Banco de Dados</h3>

```bash
    npx prisma generate
```

<h3>Executar a API</h3>

```bash
    node server.js
```
